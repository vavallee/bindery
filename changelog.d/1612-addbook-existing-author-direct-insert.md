### Fixed
- **Explicitly added books can no longer be vetoed by catalogue-sync filters** (#1612) — adding a specific work for an author already in the library only polled for a row the catalogue sync had deterministically skipped (e.g. a mis-sampled work language on heavily-translated works), so the add returned 404 "book not found after author sync — try again shortly" on every retry, forever. The synchronous direct insert introduced for just-created authors (#804) now also runs for existing authors: an explicit add fetches and persists the requested work immediately, while sync filters still govern everything the user did not explicitly pick.
